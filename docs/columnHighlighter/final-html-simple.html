About this example

Highlights the Name column when Name == '1Password' using a single AND condition. Useful for quickly drawing attention to matching values.

Configuration

columnHighlighter: {
  rules: [
    {
      conditionsContainer: [
        {
          logic: 'AND',
          conditions: [
            { columnName: 'Name', operator: 'eq', type: 'string', value: '1Password' }
          ]
        }
      ],
      targets: [
        { column: 'Name', css: { 'background-color': '#fa8072', color: '#000' } }
      ]
    }
  ]
}
